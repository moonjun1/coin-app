## 📱 앱 실행 흐름도

```
앱 시작
  ↓
[1] 스플래시/메인 화면
  ↓
[2] 대시보드 (Top 5 코인)
  ↓
[3-1] 암호화폐 시세 → 메이저코인 탭
[3-2] 암호화폐 시세 → 알트코인 탭
[3-3] 암호화폐 시세 → NFT 탭
  ↓
[4] 코인 상세 화면
  ↓
[5-1] 즐겨찾기 추가
[5-2] 메모 추가 (Dialog)
[5-3] 메모 수정 (Dialog)
[5-4] 메모 삭제 (Dialog)
  ↓
[6] 즐겨찾기 화면
  ↓
[7] 뉴스 화면
  ↓
[8] 뉴스 상세 (WebView)
  ↓
[9] 설정 화면
```

### [화면 1] 앱 시작 - MainActivity (대시보드)

**화면 구성**:

```
│                             │
│  ┌─────────────────────┐   │
│  │ BTC  비트코인         │   │
│  │ ₩85,234,567         │   │
│  │ +2.34% ↗           │   │ ← RecyclerView
│  └─────────────────────┘   │    (5개 아이템)
│                             │
│  ┌─────────────────────┐   │
│  │ ETH  이더리움         │   │
│  │ ₩3,456,789          │   │
│  │ -1.23% ↘           │   │

│                             │

│                             │
└─────────────────────────────┘
```

**동작 설명**:

1. **앱 실행 시 최초 화면**
   - MainActivity가 실행되고 DashboardFragment가 자동으로 로드됨
   - Navigation Drawer 사용 (왼쪽 상단 햄버거 메뉴 ☰)

2. **표시되는 데이터**
   - CoinGecko API에서 실시간으로 Top 5 코인 시세 가져옴

     * 심볼 (BTC, ETH 등)
     * 한글 이름 + 영문 이름 (비트코인 (Bitcoin))
     * 현재 가격 (원화)

3. **상호작용**
   - 햄버거 메뉴(☰) 클릭 → Navigation Drawer 열림
   - 코인 항목 클릭 → 해당 코인 상세 화면으로 이동

4. **구현 기술**
   - Fragment: DashboardFragment
   - Layout: RecyclerView (5개 아이템)
   - Adapter: CryptoAdapter
   - API: RetrofitClient.api.getCryptoList().take(5)
   - 비동기: lifecycleScope.launch

**코드 위치**:
- `fragments/DashboardFragment.kt` (38-56줄)

## 🎯 주요 기능 테스트 시나리오

### 시나리오 1: 코인 정보 조회 및 즐겨찾기

1. 앱 시작 → 대시보드 확인
2. 햄버거 메뉴 → "암호화폐 시세" 선택
3. "메이저코인" 탭에서 비트코인 선택
4. 상세 화면에서 정보 확인
5. 오른쪽 상단 별 아이콘 클릭 → 즐겨찾기 추가
6. 뒤로가기
7. 햄버거 메뉴 → "즐겨찾기" 선택
8. 비트코인이 목록에 표시되는지 확인

**예상 결과**: 즐겨찾기에 비트코인이 추가되어 표시됨

---

### 시나리오 2: 메모 CRUD 전체 테스트

1. 코인 상세 화면 진입
2. [메모 추가] 버튼 클릭
3. "장기 보유 추천!" 입력 후 [저장]
4. 메모가 카드에 표시되는지 확인
5. [메모 수정] 버튼 클릭
6. "단기 트레이딩용"으로 수정 후 [저장]
7. 메모 내용이 업데이트되는지 확인
8. 메모 텍스트 길게 누르기
9. 삭제 확인 Dialog에서 [삭제] 클릭
10. 메모가 사라지고 초기 상태로 돌아가는지 확인

**예상 결과**: Create → Read → Update → Delete 모두 정상 작동

---

### 시나리오 3: 탭 전환 및 NFT 개발중 메시지

1. "암호화폐 시세" 화면 진입
2. "메이저코인" 탭 확인
3. 좌우 스와이프 → "알트코인" 탭으로 전환
4. 다시 스와이프 → "NFT" 탭으로 전환
5. NFT 코인 (예: ApeCoin) 클릭
6. Toast 메시지 확인: "NFT 상세 정보는 개발중입니다 🚧"

**예상 결과**: 탭 전환 정상, NFT는 상세 화면 없음

---

### 시나리오 4: 뉴스 조회

1. 햄버거 메뉴 → "뉴스" 선택
2. 뉴스 목록 확인
3. 첫 번째 뉴스 카드 클릭
4. WebView에서 뉴스 전체 내용 확인
5. 뒤로가기 → 뉴스 목록으로 돌아감

**예상 결과**: 뉴스 목록 표시, WebView로 상세 뉴스 표시
